# Markdown 语法指南

Markdown 是一种轻量级标记语言，易于阅读和编写。

## 文本格式

**粗体**、*斜体*、~~删除线~~、`行内代码`

## 列表

无序列表：
- 苹果
- 香蕉
- 橘子

有序列表：
1. 第一步
2. 第二步
3. 第三步

## 代码块

```python
def hello(name):
    print(f"Hello, {name}!")

hello("World")
```

```javascript
function fibonacci(n) {
  if (n <= 1) return n
  return fibonacci(n - 1) + fibonacci(n - 2)
}
```
