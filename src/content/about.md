# 关于

这里是 preture 的个人站点，使用 Vue 3 + Vite 构建。

## 技术栈

- **Vue 3** — 渐进式 JavaScript 框架
- **Vite** — 现代化前端构建工具
- **Vue Router** — 客户端路由
- **markdown-it** — Markdown 渲染
- **highlight.js** — 代码语法高亮

## 关于本站

本站使用 GitHub Pages 部署，所有源码均可在 [GitHub](https://github.com/preture/preture.github.io) 上查看。

## 使用

在 `src/content/` 目录下添加 `.md` 文件即可自动生成页面。例如：

```bash
touch src/content/hello.md
```

然后在 `hello.md` 中写入内容，刷新页面即可看到新文章。
