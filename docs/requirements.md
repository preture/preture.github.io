# preture.github.io — 个人主页需求说明与页面设计方案

## 1. 项目概述

个人技术博客与项目展示站点，基于 Vue 3 + Vite 构建，通过 GitHub Pages 自动部署。内容使用 Markdown 编写，按模块归类，展示个人在编程、AI、项目实践等方面的积累与探索。

### 2.2 内容模型

```
站点
 └── 分类（Category）         — 4 个：厚积薄发 / 如虎添翼 / 身体力行 / 心之所向
      ├── 名称
      ├── 图标 / Emoji
      ├── 描述（一行简介）
      └── 主题（Topic）       — 每个分类下 3~4 个主题
           ├── 标题
           ├── 封面图（可选）
           ├── 简短摘要
           ├── 状态标签：进行中 / 计划中 / 已完成
           └── 文章（Article）— 每个主题下 N 篇文章
                ├── 标题
                ├── 发布日期
                ├── 标签（可选）
                └── Markdown 正文
```

### 2.3 页面类型

| 页面 | 说明 | URL 示例 |
|------|------|----------|
| 首页 | 个人简介 + 四个分类概览卡片 | `/#/` |
| 分类页 | 某个分类下的所有主题列表 | `/#/accumulation` |
| 主题页 | 某个主题下的所有文章列表 | `/#/accumulation/programming-foundation` |
| 文章页 | 某篇 Markdown 文章的完整内容 | `/#/accumulation/programming-foundation/setup-guide` |
| 归档页（可选） | 按时间线展示所有文章 | `/#/archive` |

## 4. 视觉设计方向

### 4.1 风格

- **基调**：简洁、清晰、偏技术感
- **配色**：
  - 主色：`#2c3e50`（深蓝灰，用于头部/标题）
  - 强调色：`#3498db`（亮蓝，用于链接/交互）
  - 背景：纯白 `#ffffff` + 浅灰 `#f5f5f5` 交替
  - 文字：深灰 `#333` / 中灰 `#666`
- **字体**：系统无衬线字体（-apple-system, Segoe UI, Roboto）
- **圆角**：4px ~ 8px，轻微圆角
- **阴影**：极淡阴影，hover 时略微加深

### 4.2 分类特有配色（可选）

## 5. 技术方案

### 5.1 当前已搭建

- Vue 3 (Composition API + `<script setup>`)
- Vite 6 构建
- Vue Router (hash 模式，兼容 GitHub Pages)
- markdown-it + highlight.js 渲染 Markdown
- GitHub Actions 自动构建部署到 GitHub Pages

### 5.2 后续实现要点

| 需求 | 方案 | 状态 |
|------|------|------|
| 分类-主题-文章 三级/四级内容结构 | 配置驱动 + 文件系统 glob 自动发现 | ✅ 已实现 |
| 分类卡片首页概览 | 首页组件渲染 4 个分类卡片，带 Emoji 和配色 | ✅ 已实现 |
| 三级结构（主题 → 文章） | 目录 `category/topic/` 下直接放 `.md` 文件 | ✅ 已实现 |
| 四级结构（主题 → 方向 → 文章） | 目录 `category/topic/subtopic/` 放 `.md` 文件 | ✅ 已实现 |
| 主题列表页 / 方向列表页 | TopicPage / SubTopicPage 自动适配显示 | ✅ 已实现 |
| 空内容占位 | 无文章时展示"功能完善中，敬请期待" | ✅ 已实现 |
| 面包屑导航 | 所有页面显示当前位置的层级路径 | ✅ 已实现 |
| 状态标签（进行中/计划中） | Markdown 前端内容 YAML frontmatter 解析 | ⏳ 待实现 |
| 文章列表按日期排序 | 基于 frontmatter 中的 `date` 字段排序 | ⏳ 待实现 |
| 响应式布局 | CSS Grid / Flexbox，移动端适配 | ✅ 已实现 |
| 代码高亮 | highlight.js 集成，支持多语言语法高亮 | ✅ 已实现 |

## 6. 后续迭代方向（v2）

- 搜索功能（客户端全文检索，Fuse.js / minisearch）
- 明暗主题切换（CSS 变量 + `prefers-color-scheme`）
- 文章 RSS（构建时生成 `feed.xml`）
- **评论系统**：[giscus](https://giscus.app)
  - 基于 GitHub Discussions，评论数据存在 repo 中
  - 访客通过 GitHub OAuth 登录评论
  - 零后端，纯静态友好
- **访问统计**：[不蒜子](http://busuanzi.ibruce.info)
  - 一行 `<script>` 嵌入，自动统计 PV/UV
  - 无需注册、无需服务端，纯前端方案
