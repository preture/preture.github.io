# Python 3 关键字详解

> Python 3.14 版本共 **35 个关键字**（`keyword.kwlist`）
> 关键字是 Python 预留的标识符，不能用作变量名、函数名或类名。

---

## 一、值关键字（3 个）

### `False` / `True` — 布尔值

```python
a = True
b = False
print(type(a))  # <class 'bool'>
print(True == 1, False == 0)  # True True
```

- `bool` 是 `int` 的子类
- 常用于条件判断

### `None` — 空值

```python
x = None
print(x is None)   # True（用 is 判断）
print(type(x))     # <class 'NoneType'>
```

- 表示"没有值"或"空"
- 函数无 `return` 时默认返回 `None`
- 判断用 `is None`，不要用 `== None`

---

## 二、逻辑运算符（3 个）

### `and` — 逻辑与

```python
True and False   # False
True and True    # True

# 短路求值：左侧为 False 时右侧不求值
0 and print("不会执行")
```

### `or` — 逻辑或

```python
True or False    # True
False or False   # False

# 短路求值：左侧为 True 时右侧不求值
1 or print("不会执行")
```

### `not` — 逻辑非

```python
not True    # False
not False   # True
not 0       # True（0 为假）
not "abc"   # False（非空字符串为真）
```

## 三、条件与循环（6 个）

### `if` / `elif` / `else` — 条件判断

```python
score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "D"
```

- `elif` 是 `else if` 的缩写
- 条件不用加括号
- `:` 结尾 + 缩进代码块

### `for` — for 循环

```python
# 遍历可迭代对象
for i in range(5):
    print(i)

for fruit in ["apple", "banana"]:
    print(fruit)

for key, value in {"a": 1, "b": 2}.items():
    print(key, value)
```

### `while` — while 循环

```python
i = 0
while i < 5:
    print(i)
    i += 1

# 死循环（需 break 退出）
while True:
    cmd = input(">> ")
    if cmd == "quit":
        break
```

### `break` — 跳出循环

```python
for i in range(10):
    if i == 5:
        break     # 立即结束循环
    print(i)      # 输出 0 1 2 3 4
```

### `continue` — 跳过本次迭代

```python
for i in range(5):
    if i == 2:
        continue  # 跳过本次，进入下一次
    print(i)      # 输出 0 1 3 4
```

## 四、函数相关（3 个）

### `def` — 定义函数

```python
def greet(name):
    return f"Hello, {name}!"

# 可设置默认参数
def power(base, exp=2):
    return base ** exp
```

### `return` — 返回值

```python
def add(a, b):
    return a + b          # 返回结果

def noop():
    return                # 返回 None

def multi():
    return 1, 2, 3        # 返回元组 (1, 2, 3)
```

- 函数遇到 `return` 立即结束
- 可以返回多个值（打包成元组）

### `lambda` — 匿名函数

```python
square = lambda x: x ** 2
print(square(5))  # 25

# 常用于高阶函数
list(map(lambda x: x * 2, [1, 2, 3]))  # [2, 4, 6]
```

- 只能写单行表达式
- 不能包含语句（不能用 `return`）

class Dog:
    def __init__(self, name):

        print(f"{self.name}: Woof!")

dog = Dog("Buddy")

## 六、异常处理（4 个）

### `try` / `except` / `finally` — 异常捕获

```python
try:
    x = 1 / 0                     # 可能抛出异常
except ZeroDivisionError:
    print("不能除以零！")
except TypeError as e:
    print(f"类型错误：{e}")
else:
    print("没有异常时执行")
finally:
    print("无论是否异常都会执行")
```

| 子句 | 说明 |
|------|------|
| `try` | 放可能出错的代码 |
| `except` | 捕获并处理异常 |
| `else` | 无异常时执行 |
| `finally` | 始终执行（清理资源） |

### `raise` — 主动抛出异常

```python
def withdraw(balance, amount):
    if amount > balance:
        raise ValueError("余额不足！")
    return balance - amount

# 也可以重新抛出
try:
    withdraw(100, 200)
except ValueError:
    print("记录日志...")
    raise  # 继续向上抛
```

### `assert` — 断言

```python
def divide(a, b):
    assert b != 0, "除数不能为 0"
    return a / b

# 等价于：
# if b == 0:
#     raise AssertionError("除数不能为 0")
```

- 用于调试，生产环境可以用 `-O` 关闭

## 八、上下文管理（1 个）

### `with` — 上下文管理器

```python
# 自动关闭文件
with open("hello.txt", "r") as f:
    content = f.read()
# 文件已自动关闭

# 同时管理多个
with open("a.txt") as f1, open("b.txt") as f2:
    pass
```

需要实现 `__enter__` 和 `__exit__` 方法。

## 九、变量作用域（2 个）

### `global` — 声明全局变量

```python
count = 0

def increment():
    global count   # 不加这行会创建局部变量
    count += 1

increment()
print(count)  # 1
```

### `nonlocal` — 声明外层变量（闭包）

```python
def outer():
    x = 10

    def inner():
        nonlocal x   # 修改外层函数的 x
        x += 1

    inner()
    print(x)  # 11
```

## 十、成员与身份判断（2 个）

### `in` — 成员判断

```python
"a" in "abc"            # True（字符串包含）
3 in [1, 2, 3]          # True（列表包含）
"x" in {"a": 1}         # False（字典判断键）
"a" in {"a": 1}         # True

# 也可以用于 for 循环
for x in range(5):
    pass
```

### `is` — 身份判断

```python
a = [1, 2, 3]
b = [1, 2, 3]
c = a

a == b    # True（值相等）
a is b    # False（不同对象）
a is c    # True（同一对象）

# None 判断必须用 is
x is None
x is not None
```

## 十一、删除操作（1 个）

### `del` — 删除变量或元素

```python
# 删除变量
x = 10
del x
# print(x)  # NameError

# 删除列表元素
lst = [1, 2, 3, 4, 5]
del lst[0]      # [2, 3, 4, 5]
del lst[1:3]    # [2, 5]

# 删除字典键
d = {"a": 1, "b": 2}
del d["a"]      # {"b": 2}
```

## 十三、生成器（1 个）

### `yield` — 生成器

```python
def countdown(n):
    while n > 0:
        yield n     # 返回一个值，暂停函数
        n -= 1

# 调用返回生成器对象
for num in countdown(3):
    print(num)  # 3 2 1

# 也可以手动迭代
gen = countdown(3)
next(gen)  # 3
next(gen)  # 2
next(gen)  # 1
```

与 `return` 的区别：`yield` 暂停函数，下次调用继续执行。

## 十四、异步编程（2 个）

### `async` — 声明异步函数

```python
import asyncio

async def fetch_data():
    print("开始获取...")
    await asyncio.sleep(1)
    print("获取完成")
    return {"data": 123}

# 运行
result = asyncio.run(fetch_data())
```

### `await` — 等待异步结果

```python
async def main():
    # 等待异步操作完成
    data = await fetch_data()
    print(data)

# await 只能在 async 函数中使用
```

## 十五、软关键字补充说明

> 以下单词在某些上下文中是关键字，但**不在** `keyword.kwlist` 中：

| 软关键字 | 说明 |
|---------|------|
| `match` | 结构模式匹配（Python 3.10+） |
| `case` | 模式匹配的分支（Python 3.10+） |
| `_` | 在 `match/case` 中表示通配符 |

示例：

```python
# Python 3.10+ match/case
def describe(value):
    match value:
        case 0:
            return "零"
        case 1 | 2 | 3:
            return "小数字"
        case _:
            return "其他"
```

### 快速判断关键字

```python
import keyword

print(keyword.kwlist)          # 所有关键字列表
print(keyword.iskeyword("if"))  # True
print(keyword.iskeyword("foo")) # False
```

### 关键字不能用作标识符

```python
# 以下都会报 SyntaxError
if = 10
def = "hello"
class = [1, 2, 3]
```

### Python 版本演化中变化的关键字

| 版本 | 变化 |
|------|------|
| 2.4 | 新增 `yield` |
| 2.5 | 新增 `with` 和 `as` |
| 2.6 | 新增 `True` `False` `None` 为关键字（此前是内置变量） |
| 3.0 | 移除 `print` 和 `exec`（变为内置函数） |
| 3.5 | 新增 `async` 和 `await` |
| 3.10 | 新增软关键字 `match` `case` `_` |
