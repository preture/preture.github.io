# Markdown 格式说明

Markdown 是一种轻量级标记语言，以纯文本形式编写，可转换为 HTML。

## 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 段落与换行

段落之间用空行分隔。行尾加两个空格再换行可实现软换行。

## 强调

```markdown
*斜体* 或 _斜体_
**粗体** 或 __粗体__
***粗斜体*** 或 ___粗斜体___
~~删除线~~
```

## 列表

### 无序列表

```markdown
- 项一
- 项二
  - 嵌套项（缩进 2 或 4 空格）
- 项三
```

### 有序列表

```markdown
1. 第一项
2. 第二项
   1. 嵌套第一项
3. 第三项
```

## 链接

```markdown
[显示文本](https://example.com)
[带标题的链接](https://example.com "标题")
<https://autolink.com>
```

## 图片

```markdown
![替代文本](https://example.com/image.png)
![带标题的图片](https://example.com/image.png "标题")
```

## 代码

### 行内代码

```markdown
使用 `print()` 函数输出。
```

### 代码块

````markdown
```python
def hello():
    print("Hello, World!")
```
````

## 引用

```markdown
> 一级引用
>
> > 二级引用
>
> 继续一级引用
```

## 分隔线

```markdown
---
***
___
```

## 表格

```markdown
| 左对齐 | 居中 | 右对齐 |
| :----- | :--: | -----: |
| 内容   | 内容 | 内容   |
| 内容   | 内容 | 内容   |
```

## 任务列表

```markdown
- [x] 已完成任务
- [ ] 未完成任务
```

## 脚注

```markdown
这是一个脚注[^1]。

[^1]: 脚注内容写在这里。
```

## 转义字符

用反斜杠转义特殊字符：

```markdown
\* 星号 \_ 下划线 \[ 方括号
```

## 扩展语法（GFM）

GitHub Flavored Markdown 额外支持：

- **删除线**：`~~文本~~`
- **表格**：如上所示
- **任务列表**：如上所示
- **自动链接**：URL 自动转为链接
- **表情符号**：`:smile:` → :smile:
