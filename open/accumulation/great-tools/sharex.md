# ShareX安装和使用

## 简介

ShareX 是一款开源免费的截图、录屏、文件分享工具。支持截图、屏幕录制、颜色拾取、OCR 文字识别、文件上传等一系列功能，并将结果一键分享到剪切板或各大图床、文件托管服务。

## 安装

### Windows

```bash
# 1. 方式一：官网下载
open https://getsharex.com

# 2. 方式二：Microsoft Store 安装
# 在 Microsoft Store 搜索 "ShareX"

# 3. 方式三：winget
winget install ShareX.ShareX

# 下载后双击安装程序，按向导完成安装即可
```

### macOS / Linux

ShareX 仅支持 Windows。macOS 替代方案：[Shottr](https://shottr.cc) 或系统自带截图工具。Linux 替代方案：[Flameshot](https://flameshot.org)。

## 核心特性

- **截图模式丰富**：全屏、窗口、区域、滚动截图、延时截图等
- **屏幕录制**：支持录制为 GIF 或 MP4
- **OCR 文字识别**：从截图中提取文字
- **颜色拾取**：取色器 + 屏幕放大镜
- **自动上传**：截图后自动上传到 Imgur、Google Photos、GitHub、S3、FTP 等 80+ 目标
- **快捷动作**：截图后自动执行添加水印、调整尺寸、模糊处理等操作
- **快捷键工作流**：自定义快捷键完成截屏 → 编辑 → 上传 → 复制链接全流程
- **历史记录**：所有截图和上传记录本地管理，随时回看

## 使用场景

### 1. Bug 截图反馈

截图时自动添加标注（箭头、矩形、文字、模糊等），标注后一键上传到图床并自动复制链接，可粘贴到 Jira、GitHub Issues、飞书等工单系统。

### 2. 文档截图教程

使用滚动截图捕获长网页或完整聊天记录，省去多张截图拼接的麻烦，适合制作操作指南和教程。

### 3. OCR 文字提取

截取图片中的文字区域，按 `OCR` 快捷键即可识别出文字内容到剪切板，不再需要手动抄录。

### 4. 颜色取色

前端开发或设计中取色时，用颜色拾取功能直接获取屏幕任意位置的颜色值 (HEX/RGB/HSL)，无需打开 PS。

### 5. 快速文件分享

本地文件右键 → ShareX → 自动上传到指定目标并复制分享链接，比开网盘拖文件快得多。

### 6. 屏幕录制演示

快速录制操作步骤为 GIF，适合在 PR 中展示 UI 交互或给同事演示操作流程，无需启动专业录屏软件。

## 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Print Screen` | 截图区域选择 |
| `Ctrl + Print Screen` | 截取全屏（隐藏窗口） |
| `Shift + Print Screen` | 截取活动窗口 |
| `Ctrl + Shift + Print Screen` | 屏幕录制 |
| `Shift + 双击任务栏图标` | 打开截图历史记录 |
| `Shift + 右键任务栏图标` | 快速选择上传文件 |

## 注意事项

- ShareX 开源免费，无广告无内购
- 在「应用程序设置 > 上传目标」中配置图床/文件服务后，上传功能才能真正发挥价值
- 建议设置「截图后动作」为「打开编辑器」+「保存图像到文件」+「上传图像」，形成一条完整工作流
- 默认快捷键与 Windows 系统截图有冲突时可在「热键设置」中修改
