# 剪映 (CapCut)安装和使用

## 简介

剪映（海外版名 CapCut）是字节跳动推出的一款免费视频剪辑软件，提供桌面端和移动端版本。功能覆盖从基础剪辑到专业调色、特效、字幕生成等场景，内置海量素材库（贴纸、滤镜、转场、音乐），上手门槛极低。

## 安装

### Windows / macOS

```bash
# 1. 官网下载桌面版
open https://www.capcut.cn

# 2. 双击安装程序，按向导完成安装
```

### iOS / Android

在 App Store 或各大应用商店搜索「剪映」下载即可。

### 网页版

直接访问 https://www.capcut.com/editor 在线使用，无需安装。

## 核心特性

- **智能字幕**：自动识别视频语音生成字幕，支持多语言和手动调整
- **海量模板**：一键套用抖音/ TikTok 热门视频模板
- **素材库**：内置音乐、音效、贴纸、滤镜、转场、特效
- **关键帧动画**：自定义位置、缩放、旋转、透明度动画
- **蒙版与混合模式**：画中画、抠像、叠加效果
- **曲线变速**：自定义视频变速曲线，实现丝滑慢动作或快放
- **智能工具**：AI 抠像、文本转语音、智能调色
- **多轨道编辑**：主轨 + 画中画 + 音频 + 文本 + 贴纸多轨叠加

## 使用场景

### 1. 短视频创作

快速剪辑抖音、TikTok、小红书等平台的短视频，配合模板和热门音乐快速出片。

### 2. 视频字幕制作

使用智能字幕功能，一键识别视频中的语音并生成时间轴字幕，支持导出 SRT 文件。

### 3. 教程录制与剪辑

录制屏幕操作后导入剪映，添加标注、缩放、转场和配音，制作清晰的教学视频。

### 4. Vlog 生活记录

剪辑旅行、日常 Vlog，利用滤镜、转场和背景音乐增强视频氛围感。

### 5. 社交媒体广告

制作产品展示或促销短视频，配合文字动画和特效在短时间内吸引注意力。

### 6. 音频处理

分离视频中的背景音乐和人声，或使用文本转语音功能生成配音。

## 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Space` | 播放/暂停 |
| `Ctrl + B` | 分割片段 |
| `Ctrl + C / V` | 复制/粘贴片段 |
| `Ctrl + Z / Y` | 撤销/重做 |
| `Ctrl + D` | 复制选中片段 |
| `Del` | 删除选中片段 |
| `Ctrl + Shift + E` | 导出 |
| `+ / -` | 放大/缩小时间轴 |
| `↑ / ↓` | 上下移动选中轨道 |

## 注意事项

- 桌面版部分高级功能（如 4K 导出、Pro 调色）需要剪映专业版会员
- 素材库内的部分音乐和贴纸有版权限制，商用需注意授权
- 工程文件建议定期备份，剪映的自动保存路径可在设置中查看
- 导出时可根据平台需求选择分辨率（1080p / 4K）和码率
